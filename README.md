# myHexoBlog

## Install
### Node.js
sudo apt-get install npm

### Git
sudo apt-get install git

### Hexo
npm instal -g hexo-cli


## Usage
hexo g # 或者hexo generate
hexo s # 或者hexo server，可以在http://localhost:4000/ 查看

常用简写

hexo n == hexo new

hexo g == hexo generate

hexo s == hexo server

hexo d == hexo deploy

常用组合

hexo d -g #生成部署

hexo s -g #生成预览

## Deploy to github.io
./deploy.sh
